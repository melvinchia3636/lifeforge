- <Commit hash="340a713993368ccb394eec73f372f1428b7e3daa" />: Moved

  to be manually wrapped around each module page. Also, module title and icon
  are now passed via context provider in this level instead of manually defining
  them in each module page. However, explicit definition is still supported for
  special use cases.

  ```tsx
  // Before
  return (
    <ModuleWrapper>
      <ModuleHeader icon="tabler:calendar" title="Calendar" />
      ...
    </ModuleWrapper>
  )

  // After
  return (
    <>
      <ModuleHeader />
      ...
    </>
  )

  // Or, for explicit definition
  return (
    <>
      <ModuleHeader icon="tabler:cube" title="Subpage" />
      ...
    </>
  )
  ```
